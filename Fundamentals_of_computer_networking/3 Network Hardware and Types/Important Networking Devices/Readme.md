## 2Ô∏è‚É£ Repeater

- When two devices are very far apart, the signal becomes **weak as distance increases**.
- A **Repeater** is used to **regenerate and strengthen weak signals**.
- It receives the weak signal and retransmits it at a higher strength.
- Common use:
  - Boosting **Wi-Fi signals**
  - Long-distance network communication

---

## 3Ô∏è‚É£ Hub

- A **Hub** is a special type of repeater placed at the **center of a network**.
- All devices are connected to the hub.
- It:
  - Receives data from one device
  - Broadcasts it to **all connected devices**
- Only the intended device uses the data; others ignore it.
- ‚ùå This causes:
  - Bandwidth wastage
  - Data collisions
- Hubs are rarely used today.

## 4Ô∏è‚É£ Switch

- A **Switch** is a smarter version of a hub.
- It has the same physical structure as a hub but works more efficiently.
- Instead of broadcasting data:
  - It checks the **destination device**
  - Sends data **only to that device**
- ‚úÖ Reduces bandwidth loss and collisions
- Widely used in modern networks

---

## 5Ô∏è‚É£ Router

- A **Router** connects different networks together.
- It works like a switch but also:
  - Routes Internet traffic
  - Decides where data packets should go
- Modern routers also provide:
  - **Wireless (Wi-Fi)** connectivity
- A router connects your **local network to the Internet**.

## 6Ô∏è‚É£ Bridge

- A **Bridge** is used to **connect two or more network segments**.
- It helps reduce traffic by dividing a large network into smaller parts.
- Bridges work at the **Data Link Layer (Layer 2)**.

## 8Ô∏è‚É£ Access Point (AP)

  - Campuses
- Multiple access points can be installed in different rooms.
